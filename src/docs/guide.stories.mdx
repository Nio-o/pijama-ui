# О документе

QIWI Guide — это развивающийся набор рекомендаций и правил по оформлению интерфейсов QIWI в заданном стиле. Документ содержит базовые рекомендации по дизайну и корректной подготовке макетов к разработке.

## Кому полезен

— дизайнеру интерфейсов,

— контент-менеджеру,

— копирайтеру,

— продуктовому менеджеру,

— бренд-менеджеру,

— промо-дизайнеру,

— разработчику,

— аналитику.

## Назначение

— поддерживать единый стиль,

— стандартизировать дизайн-решения,

— сохранять и развивать заданный уровень качества,

— ускорять поиск дизайн-решений и погружения в дизайн-систему продукта.

## Как использовать

При создании интерфейса лучше использовать готовые решения, описанные в гайде.

Если задача принципиально новая и подходящих решений пока нет, допустимо разработать новое решение, соответствующее описанному в документе стилю.

Не стоит вводить новые стили для уже существующих функциональных элементов — чекбоксов, кнопок, текстовых блоков.

Рекомендуем следовать заданной сетке сайта и правилам композиционного решения страниц. Это поможет соблюдать согласованность.

Инструкции по подготовке контента следует применять во всех случаях, описанных в документе.
